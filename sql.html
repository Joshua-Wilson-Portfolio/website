SQL Projects

In this project, I used SQL to clean, transform, and analyse Spotify streaming data to uncover trends in user behaviour and artist performance. This included:

  • Removing duplicate rows and formatting inconsistencies across multiple tables
  • Using joins, window functions, and aggregate queries to identify top artists by region and genre
  • Calculating user retention and engagement metrics across time periods
  • Creating a clean, analysis-ready dataset suitable for BI tools like Tableau and Power BI

This project highlights my ability to write efficient queries, structure datasets for analysis, and extract business value from raw relational data.
