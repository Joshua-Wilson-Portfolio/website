Project Overview

This Python project analyzes quarterly apparel sales across Australian states. It aims to extract actionable insights through wrangling, summarization, and visualization techniques.

Sales Analysis – Python Project

This project explores apparel sales data in Australia during Q4 2020. Below is a step-by-step breakdown of the workflow including data cleaning, exploration, and insights using Python.

1. Load Dataset

The dataset is loaded using pd.read_csv().

2. Preview Data

Using df.head() and df.tail() to view the first and last few rows.

3. DataFrame Info

Checking structure, memory usage, and null values.

4. Data Types

Inspecting data types to determine what preprocessing is required.

5. Null Values Check

Ensuring no missing values are present using df.isnull().sum().

6. Duplicates Check

Checking for and dropping duplicate records using df.duplicated() and drop_duplicates().

7. Convert Date Column

Converted the Date column to datetime using pd.to_datetime().

8. Boxplot of Sales

Visualizing distribution and identifying outliers in sales.

9. Summary Statistics

Basic statistical measures such as mean, median, and standard deviation.

10. Weekly Sales Trend

Visualizing sales performance over time using line charts.

11. Sales by State and Group

Segmenting total sales by state and age demographic groups.

12. Group-wise Sales by State

Evaluating sales contributions from each demographic across states.
